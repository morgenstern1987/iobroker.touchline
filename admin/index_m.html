info_outline Hinweise zur Legacy API
Dieser Adapter verwendet ausschließlich die lokale CGI-API.
  • Nur mit dem Roth Touchline Legacy Controller kompatibel
  • Keine Cloud-Verbindung – vollständig lokale Kommunikation
  • Controller und ioBroker müssen im selben Netzwerk sein
  • Neue Touchline SL / Touchline+ werden nicht unterstützt
